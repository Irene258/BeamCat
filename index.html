使用指南

下载 BeamCat 后，按照以下步骤快速上手：

  1. 安装并启动软件
  2. 选择眼动或手势控制模式
  3. 绑定快捷键，调整灵敏度
  4. 开始智能交互！

常见问题

如何调整鼠标灵敏度？

在“设置”中选择普通 / Silver Line / Gold Zoo 模式。

是否支持 Mac？

目前仅支持 Windows 10/11，Mac 版本开发中。
